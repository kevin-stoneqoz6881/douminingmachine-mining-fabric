## What it does

DouMiningMachine is a Fabric-based Minecraft automation mod aimed at cutting down the repetitive side of mining and digging. It provides three dedicated machines that can gather blocks, route outputs, and adapt to different excavation styles in survival or modded setups.

At its core, the mod is about configurable mining control. You can tune range and speed, rely on coal as fuel, and apply filtering for ores or other materials. It also offers both same-layer digging and full-chunk excavation, which gives players more flexibility in how an area is cleared.

## Main Features

- Three purpose-built machines for mining and excavation tasks
- Adjustable range for small jobs or larger dig zones
- Speed settings to control how quickly each machine works
- Coal-powered operation for fuel-based in-game use
- Automatic chest output for mined blocks and collected resources
- Ore and material filtering for targeted drop separation
- Same-layer digging mode for flat, controlled removal
- Full-chunk digging mode for wider automated clearing
